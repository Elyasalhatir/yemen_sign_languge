# Design Notes for PowerPoint

## Color Scheme:
- Primary: #00f2ea (Cyan)
- Secondary: #ff0050 (Pink/Red)
- Background: #0a0a12 to #1a1a2e (Dark gradient)
- Text: White (#FFFFFF)
- Accent: #f1c40f (Gold)

## Fonts:
- Arabic: Cairo or Tajawal
- English: Outfit or Inter
- Code: Fira Code

## Animations:
- Slide transitions: Morph or Fade
- Text: Appear or Fade
- Charts: Wipe from left

## Tips:
- Keep each slide focused on one idea
- Use large fonts (min 24pt)
- Include visual diagrams
- Practice timing (2-3 min per slide)
- Prepare for Q&A: 5-10 extra technical slides
